# 8BitALU
Digital Logic Design 8 Bit ALU
Description: 8-Bit ALU with 7 features ( AND, OR, XOR, APass, BPass, Addition and Subtraction) 
User inputs the Opcode (3-bits) followed by A then B (8-bits). The 8-Bit ALU will output the result as 3 hexadecimal numbers and an overflow flag for Addition and Subtraction features

# Features & Opcode
<pre>AND: 000</pre>
<pre>OR: 001</pre>
<pre>XOR: 010</pre>
<pre>APass: 011</pre>
<pre>BPass: 100</pre>
<pre>Addition: 101</pre>
<pre>Subtraction: 110</pre>
